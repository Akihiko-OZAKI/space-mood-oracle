# 🚀 データベース作成 - 実行手順

## ⚡ 最も簡単な方法（推奨）

### 方法1: 一括実行SQLを使う ⭐

1. **ファイルを開く**: `【一括実行】DB作成+テーブル作成.sql`

2. **SQL内容を全部コピー**

3. **データベース管理ツールで実行**:
   - **phpMyAdmin**: 
     - SQLタブを開く
     - コピーしたSQLをペースト
     - 「実行」ボタンをクリック
   
   - **MySQL Workbench**:
     - クエリエディタを開く
     - コピーしたSQLをペースト
     - ⚡（実行）ボタンをクリック

4. **完了！** ✅
   - データベース `space_mood_oracle` が作成される
   - 3つのテーブルも同時に作成される

---

### 方法2: コマンドラインで実行

```bash
mysql -u ユーザー名 -p < "【一括実行】DB作成+テーブル作成.sql"
```

パスワードを入力して実行。

---

## 📋 実行内容

このSQLファイルを実行すると：

1. ✅ **データベース作成**
   - データベース名: `space_mood_oracle`
   - 文字コード: `utf8mb4`

2. ✅ **テーブル作成**（3つ）
   - `google_trend_data` - Googleトレンドデータ
   - `twitter_trend_data` - Twitterトレンドデータ
   - `daily_mood_judgment` - 日次判定結果

---

## ✅ 実行後の確認

SQLを実行後、以下のSQLで確認できます：

```sql
-- データベースが作成されたか確認
SHOW DATABASES;

-- データベースを選択
USE space_mood_oracle;

-- テーブル一覧を確認
SHOW TABLES;

-- 期待される結果:
-- google_trend_data
-- twitter_trend_data
-- daily_mood_judgment
-- （既存のテーブルも表示される可能性があります）
```

---

## 🆘 エラーが出た場合

### "Access denied"

→ データベースへのアクセス権限がない  
→ 管理者権限でログインするか、権限のあるユーザーで実行

### "Database already exists"

→ 既にデータベースが存在する  
→ 問題ありません。そのまま使用できます

### "Table already exists"

→ 既にテーブルが存在する  
→ 問題ありません。`CREATE TABLE IF NOT EXISTS` なのでスキップされます
