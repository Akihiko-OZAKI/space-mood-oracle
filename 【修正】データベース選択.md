# ⚠️ データベース選択の修正

## 📋 重要な変更

**PlanetScaleの無料プランは2024年3月に終了しました。**

現在、最低プランは月額$39からとなっています。

---

## 🏆 新しい推奨: TiDB Cloud ⭐

### なぜ TiDB Cloud か？

1. ✅ **完全無料プランあり**
   - Developer Tier（無料）
   - 制限ありだが本番環境に使用可能

2. ✅ **MySQL互換**
   - コード変更不要
   - 既存のスキーマがそのまま使える

3. ✅ **引継ぎサマリーでも推奨**
   - プロジェクトの元の推奨データベース

4. ✅ **本番環境対応**
   - 分散データベース
   - 高パフォーマンス

### 無料枠（Developer Tier）

- **ストレージ**: 25GB行ストレージ + 25GB列ストレージ
- **リクエスト**: 250万リクエストユニット（RUs）/月
- **データベース**: 複数作成可能
- **期限**: 無料枠あり（詳細要確認）

---

## 🔄 代替案

### オプション1: TiDB Cloud ⭐ 最推奨

- MySQL互換
- 無料プランあり
- 引継ぎサマリーで推奨

**セットアップ:**
1. https://tidbcloud.com/ にアカウント作成
2. Developer Tier（無料）を選択
3. クラスターを作成
4. 接続情報を取得

### オプション2: Supabase

- PostgreSQL（MySQLではない）
- 完全無料プランあり
- **注意**: スキーマ変更が必要

### オプション3: AWS RDS

- MySQL対応
- 12ヶ月間無料
- **注意**: 12ヶ月後は有料
