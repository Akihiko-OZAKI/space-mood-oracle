# ⚡ クイックスタート - 3ステップで完了

## 📋 ステップ1: マイグレーション実行（5分）

### 最も簡単な方法

1. **ファイルを開く**
   ```
   マイグレーション実行SQL.sql
   ```

2. **SQLをコピー**
   - ファイル全体をコピー（CREATE TABLE文3つ）

3. **データベースで実行**
   - phpMyAdmin: SQLタブ → ペースト → 実行
   - MySQL Workbench: クエリエディタ → ペースト → 実行

4. **確認**
   ```sql
   SHOW TABLES LIKE '%trend%';
   SHOW TABLES LIKE '%mood%';
   ```
   → 3つのテーブル名が表示されればOK ✅

## 📋 ステップ3: APIテスト（2分）

### ブラウザの開発者ツールで

1. ブラウザで `http://localhost:3000` を開く
2. F12 で開発者ツールを開く
3. Consoleタブで以下を実行:

```javascript
fetch('/api/trpc/trends.getTodayMood')
  .then(r => r.json())
  .then(data => {
    console.log('✅ 成功！', data);
  })
  .catch(err => {
    console.error('❌ エラー', err);
  });
```

**期待される結果:**
```json
{
  "result": {
    "data": {
      "success": true,
      "judgment": {
        "judgment": "good",
        "score": 0.45,
        ...
      }
    }
  }
}
```

→ この結果が返ってくれば完了！🎉

## 🆘 エラーが出た場合

| エラー | 原因 | 解決策 |
|--------|------|--------|
| "Table doesn't exist" | マイグレーション未実行 | ステップ1を実行 |
| "Cannot connect" | DATABASE_URL未設定 | .envファイルを作成 |
| 404エラー | サーバー未起動 | ステップ2を実行 |
