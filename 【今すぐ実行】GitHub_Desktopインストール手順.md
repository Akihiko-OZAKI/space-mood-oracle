# 🚀 GitHub Desktop インストール手順

## 📥 ステップ1: ダウンロード

1. **ブラウザで以下のURLを開く**
   - https://desktop.github.com/
   - または検索エンジンで「GitHub Desktop」と検索

2. **「Download for Windows」ボタンをクリック**
   - 大きな緑色のボタンがあります
   - ダウンロードが自動的に開始されます

---

## 💿 ステップ2: インストール

1. **ダウンロードされたファイルを実行**
   - 通常は `GitHubDesktopSetup.exe` というファイル名
   - ダウンロードフォルダにあります

2. **インストーラーが起動**
   - 「GitHub Desktop Setup」ウィンドウが開きます

3. **インストールを実行**
   - 「Install」ボタンをクリック
   - インストールが完了するまで待ちます（数分かかります）

4. **完了**
   - 「Launch GitHub Desktop」にチェックが入っている
   - 「Finish」ボタンをクリック

---

## 🎯 ステップ3: 初回セットアップ

### 初回起動時

1. **GitHub Desktopが起動**
   - 初回セットアップ画面が表示されます

2. **GitHubアカウントでサインイン（推奨）**
   - 「Sign in to GitHub.com」をクリック
   - ブラウザでGitHubにログイン
   - 「Authorize desktop」をクリック

   **※GitHubアカウントがない場合**：
   - 「Skip this step」をクリック
   - 後でサインインできます

3. **設定を完了**
   - 名前とメールアドレスを入力（任意）
   - 「Continue」をクリック

---

## 📁 ステップ4: リポジトリを追加

1. **「File」→「Add Local Repository」をクリック**
   - または、「Let's get started!」画面から「Add」ボタン

2. **フォルダを選択**
   - 「Choose...」ボタンをクリック
   - 以下のフォルダを選択：
   ```
   H:\AI_study\228_宇宙パワー_V1\space_mood_oracle_v2.1\space_mood_oracle
   ```

3. **「Add repository」をクリック**
   - リポジトリが追加されます

---

## ✅ ステップ5: 変更を確認

リポジトリを追加すると：

1. **左側に「Changes」タブが表示されます**
   - 「8 changed files」と表示されるはずです

2. **変更されたファイルが一覧表示されます**
   - すべてのファイルが自動的に選択されています

---

## 💬 ステップ6: コミット・プッシュ

1. **下部の「Summary」にコミットメッセージを入力**
   ```
   🔒 セキュリティ修正: 接続文字列から実際の認証情報を削除
   
   - 実際のDB接続文字列（パスワード含む）をプレースホルダーに置換
   - .gitignoreに*.envパターンを追加して.envファイルを保護
   - GitHubの警告に対応
   - TiDB Cloudのパスワードをリセットして無効化
   - Renderの環境変数を新しいパスワードに更新
   ```

2. **「Commit to main」をクリック**
   - コミットが完了します

3. **上部の「Push origin」をクリック**
   - GitHubにプッシュされます
   - 完了！

---

## 🆘 トラブルシューティング

### インストールが途中で止まる

- インターネット接続を確認
- ウイルス対策ソフトを一時的に無効化して再試行

### リポジトリが見つからない

- フォルダパスが正しいか確認
- 既にGitリポジトリとして初期化されているか確認

### GitHubにサインインできない

- 「Skip this step」でスキップ可能
- 後で「File」→「Options」→「Accounts」からサインインできます

---

3. ✅ GitHub Desktopを起動
4. ✅ リポジトリを追加
5. ✅ コミット・プッシュ

**所要時間：約5分** ⏱️
