## 📊 使用しているデータの入手元と内容

### 🌍 宇宙天気データ

**データソース:** NOAA Space Weather Prediction Center

- **正式名称:** National Oceanic and Atmospheric Administration（米国海洋大気庁）
- **機関:** 米国商務省の下にある連邦政府機関
- **信頼性:** 世界標準の宇宙天気データを提供する公式機関
- **URL:** https://services.swpc.noaa.gov/json
- **認証:** 不要（完全無料、オープンデータ）
- **データ内容:**
  - Kp指数（地磁気活動: 0〜9）
  - 太陽フレア（Xクラス、Mクラス）
  - プロトンフラックス
  - 太陽風速度

#### 📈 各データ項目の詳細説明

##### 1. Kp指数（地磁気活動: 0〜9）

**素人向け説明:**
地球の磁場がどれだけ「揺れているか」を示す指数です。太陽から来る粒子が地球の磁場を乱すことで、この値が上がります。天気予報の「気圧」のようなもので、宇宙天気の基本的な指標です。

**理論的背景:**
- **測定方法:** 世界中の13の観測所で地磁気の変動を測定し、3時間ごとに平均値を算出
- **スケール:** 0（静穏）から9（極端な嵐）まで、対数スケールで表現
- **物理的意味:** 太陽風（太陽から放出される荷電粒子の流れ）が地球磁気圏に与える影響の強度
- **人間への影響の可能性:**
  - 強い地磁気活動（Kp 5以上）は、生体リズムや睡眠パターンに影響を与える可能性が研究で示唆されている
  - 一部の研究では、地磁気の変動と人間のメンタルヘルス、特にうつ病や不安の関連性が報告されている
  - メラトニン（睡眠ホルモン）の分泌に影響を与える可能性がある

##### 2. 太陽フレア（Xクラス、Mクラス）

**素人向け説明:**
太陽表面で起こる巨大な「爆発」です。太陽の表面で磁場がねじれて、突然エネルギーを放出する現象で、X線や紫外線を大量に放出します。クラスは強度を表し、C（小規模）→ M（中規模）→ X（大規模）の順に強くなります。

**理論的背景:**
- **分類システム:** GOES衛星が観測するX線の強度で分類
  - **Cクラス:** 10⁻⁶ W/m²以上（小規模）
  - **Mクラス:** 10⁻⁵ W/m²以上（中規模、通信障害の可能性）
  - **Xクラス:** 10⁻⁴ W/m²以上（大規模、広範囲の通信障害、停電の可能性）
- **物理的意味:** 太陽の磁場エネルギーが突然解放される現象。コロナ質量放出（CME）を伴うことが多い
- **発生頻度:** Xクラスフレアは年間数回、Mクラスは月に数回程度
- **人間への影響の可能性:**
  - 高エネルギーのX線や紫外線は、大気圏で吸収されるため直接的な健康影響は限定的
  - しかし、地磁気嵐を引き起こし、間接的に生体リズムに影響を与える可能性
  - 航空機の乗客や宇宙飛行士は、高高度での被ばくリスクが増加

##### 3. プロトンフラックス

**素人向け説明:**
太陽から飛んでくる高エネルギーの陽子（プロトン）の量です。太陽フレアが発生すると、大量の陽子が地球に向かって飛んできます。これは「宇宙線」の一種で、放射線に近い性質を持ちます。

**理論的背景:**
- **測定方法:** GOES衛星が10 MeV（メガエレクトロンボルト）以上の高エネルギー陽子を検出
- **単位:** pfu（proton flux unit、陽子フラックス単位）。1 pfu = 1陽子/cm²/秒/ステラジアン
- **Sスケール:** プロトンフラックスの強度を5段階で分類
  - S1（Minor）: 10 pfu以上
  - S2（Moderate）: 100 pfu以上
  - S3（Strong）: 1,000 pfu以上
  - S4（Severe）: 10,000 pfu以上
  - S5（Extreme）: 100,000 pfu以上
- **人間への影響の可能性:**
  - 高高度（航空機、宇宙）での被ばくリスクが増加
  - 一部の研究では、プロトンフラックスの増加と生体リズムの乱れ、免疫機能への影響が示唆されている
  - 極端な場合（S5レベル）、地上でも影響を受ける可能性がある

##### 4. 太陽風速度

**素人向け説明:**
太陽から吹き出している「風」の速さです。ただし、これは空気の風ではなく、太陽から放出される荷電粒子（主に電子と陽子）の流れです。通常は時速約150万kmですが、太陽フレアが発生すると時速300万km以上になることもあります。

**理論的背景:**
- **測定方法:** ACE衛星やDSCOVR衛星が太陽風の速度、密度、磁場強度をリアルタイムで観測
- **通常値:** 約400 km/s（時速約144万km）
- **高速太陽風:** 600 km/s以上（時速約216万km以上）になると、地磁気活動が活発化
- **物理的意味:** 太陽のコロナ（外層大気）から常に流出しているプラズマ（電離したガス）の流れ
- **人間への影響の可能性:**
  - 太陽風自体は直接的な影響は少ないが、地球磁気圏との相互作用で地磁気嵐を引き起こす
  - 高速太陽風が続くと、Kp指数が上昇し、間接的に生体リズムに影響を与える可能性
  - 通信障害やGPSの精度低下を引き起こし、間接的に人間の活動に影響

---

#### 🔬 なぜこれらのデータが「集合意識」と関連するのか？

**理論的仮説:**
1. **生体リズムへの影響:** 地磁気活動の変動が、人間の体内時計やホルモン分泌に影響を与える可能性
2. **睡眠の質:** Kp指数が高い日は、睡眠の質が低下し、それが日中の気分に影響する可能性
3. **統計的相関:** 過去のデータ分析により、宇宙天気と集合的な感情スコアの間に相関が見られる場合がある

**注意:** これは科学的に完全に証明されたものではなく、統計的な相関関係に基づく推論モデルです。本システムは「もし宇宙の影響だけで気分が決まるとしたら」という仮想的なシナリオを提示しています。

## 🏗️ システムアーキテクチャ

### GitHub → Render → Vercel の関係性

```
📦 GitHub
  ↓ (push)
⚙️ Render (バックエンドAPI)
  ↓ (API呼び出し)
✨ Vercel (フロントエンド)
```

#### 📦 GitHub
- **役割:** コードリポジトリ
- **動作:** mainブランチにpush

#### ⚙️ Render
- **役割:** バックエンドAPI
- **技術:** Node.js / tRPC
- **動作:** 自動デプロイ（GitHub pushを検知）

#### ✨ Vercel
- **役割:** フロントエンド
- **技術:** React / Vite
- **動作:** 自動デプロイ（GitHub pushを検知）

**💡 自動デプロイ:** GitHubにpushすると、RenderとVercelが自動的に最新コードをデプロイします

## 🚀 システムの仕組み

過去のデータから推論モデルを学習し、今日の「宇宙だけから見た集合意識」を推定します。

- 過去90日分のHacker Newsデータと宇宙天気データの相関を学習
- 今日の宇宙天気データから、集合意識スコアを予測
- 「もし人類の気分が宇宙だけで決まるとしたら、今日はどう見えるか？」を表示

## 📝 まとめ

1. **データソース:** NOAA（宇宙天気）とHacker News（集合意識）の無料APIを使用

3. **自動化:** pushベースの自動デプロイで運用を簡素化
4. **コンセプト:** 宇宙の影響で今日の不調を説明する、ユニークな視点
