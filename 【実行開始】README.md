## 🆘 トラブルシューティング

### マイグレーションエラー

**問題**: "Table already exists"
- **解決**: 既にテーブルが存在するので問題なし。そのまま進めてOK

**問題**: "Access denied"
- **解決**: データベースのユーザー権限を確認

### サーバー起動エラー

**問題**: "Cannot find module"
- **解決**: `npm install --legacy-peer-deps` を実行

**問題**: "DATABASE_URL is not set"
- **解決**: `.env` ファイルを作成して `DATABASE_URL` を設定

### APIエラー

**問題**: 404エラー
- **解決**: サーバーが起動しているか確認。tRPCのエンドポイントパスを確認

**問題**: "Table doesn't exist"
- **解決**: マイグレーションが未実行。ステップ1を実行
